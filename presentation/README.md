# Severino Presentation System

The presentation system extends the Joe Severino brand into 16:9 editorial decks.
It uses Inter, the canonical navy palette, the official JS mark, claim-led titles,
flat evidence surfaces, direct labels, and a quiet numbered footer.

`system.json` is the machine-readable source for future presentation generation.
The first implementation is `eu-ai-act-initial-statement.pptx`.
